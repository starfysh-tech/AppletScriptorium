# MTN Ghana's Girls Code Hackathon Crowns Winners

The final event of MTN Ghana's 30-Hour Girls Code Hackathon concluded with Locked In emerging as the overall winner for their innovative project in fintech and cybersecurity.

## Overview

The hackathon brought together 100 young women from tertiary institutions across Ghana to develop tech-driven solutions to real-world challenges.
